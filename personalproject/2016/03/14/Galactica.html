Galactica

Mar 14, 2016

Galactica is my first published Android game. It is the result of my personal project at Teesside University “Procedural Content Generation oriented on Gameplay Features”, to show my work about that theme I made a game and finally I decided to publish it on the Google Play Store.



You can download it for free here:

https://play.google.com/store/apps/details?id=org.cocos2dx.Galactica

It is done using Cocos2D-x as framework, the language is C++ and you can see the source code here:

https://github.com/JaPardoRobles/Galactica

Enjoy it!
